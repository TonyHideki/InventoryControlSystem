# 在庫管理システム (CLI版)

Javaで実装したシンプルな **在庫管理システム（CLI版）** です。
商品登録・検索・更新・削除・一覧表示を行い、データは CSV ファイルに保存されます。
ポートフォリオとして、オブジェクト指向設計、ファイル入出力、CLI 操作の理解を示すために作成したプロジェクトです。

---

## 🚀 機能一覧
- **商品一覧表示**：登録済みの商品を表形式で表示
- **商品登録**：ID / 名称 / 在庫数 / 単価 を入力して新規追加
- **商品検索**：ID もしくは商品名（部分一致）で検索
- **商品更新**：既存商品の名称・在庫数・単価を変更
- **商品削除**：指定した ID の商品を削除
- **CSV 保存 / 読み込み**：アプリ終了時にデータを保存し、次回起動時に自動で読み込み

---

## 📁 プロジェクト構成
```
InventoryControlSystem/
 ├─ inventory/
 │   ├─ Product.java        // 商品クラス
 │   ├─ Inventory.java      // 在庫管理クラス
 │   ├─ FileManager.java    // CSV保存・読み込み
 │   └─ InventoryApp.java   // CLIアプリのメイン
 └─ products.csv            // 保存されるデータ
```

---

## 🛠 使用技術
- **Java 17**（推奨）
- **CLI操作**（Scanner を使った入力処理）
- **CSV入出力**（FileManager クラスで管理）

---

## ▶️ 実行方法
### 1. コンパイル
```
javac inventory/*.java
```

### 2. 実行
```
java inventory.InventoryApp
```

### 3. データファイルについて
- アプリ終了時に `products.csv` として保存されます
- 起動時に自動読み込みされます

## 📘 設計（UML / クラス設計）
本システムは、以下の責任分担で構成されています：

- **Product**：商品データモデル
- **Inventory**：商品リストの登録・検索・更新・削除を管理
- **FileManager**：CSV ファイル保存・読み込み処理
- **InventoryApp**：ユーザーインターフェイス（CLI）

オブジェクト指向の基本である「単一責任の原則」に沿った構成となっています。
